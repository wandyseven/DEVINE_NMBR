Devine le nombre

Voici les instructions

Bienvenue dans le jeu "Devine le Nombre" ! Voici les règles :

  1. Le jeu choisit un nombre aléatoire entre 1 et 100.
  2. Vous avez 10 essais pour trouver ce nombre.
  3. Après chaque tentative, vous recevrez un indice : "C'est plus !" si le nombre est trop petit, ou "C'est moins !" si le nombre est trop grand.
  4. Si vous trouvez le bon nombre, vous gagnez ! Sinon, si vous dépassez la limite d'essais, vous perdez.

Bonne chance !

Pour jouer il faut écrire un nombre entre 1 et 100
